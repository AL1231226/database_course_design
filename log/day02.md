遇到的问题

**1. 前端如何访问后端的问题**

前后端分离开发中，前端 Vue 项目与后端 Spring Boot 运行在不同端口上，直接请求会触发浏览器的跨域安全策略。

**2. 方法如何和数据库的操作方法对应**

在 MyBatis 框架中，Mapper 接口中定义的方法需要与 XML 映射文件中的 SQL 语句建立对应关系。

**3. 前端访问错误，配置路由问题**

前端页面访问时出现 404 或空白页，排查后发现是路由配置问题。Vue Router 中需要正确定义组件与 URL 路径的对应关系。
